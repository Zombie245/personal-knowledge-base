# Personal Knowledge Base & Catalog

Легкий, швидкий та повністю автономний веб-додаток для структурування інформації, ведення особистого каталогу пристроїв/сервісів та керування приватною базою знань.

### Ключові можливості:
* **FastAPI & SQLite (WAL mode):** Висока швидкодія та стабільна робота бази даних всередині контейнера без блокування процесів читання під час запису.
* **Автономний старт (Zero-Configuration):** Додаток готовий до роботи одразу після клонування з Git — безпечний секретний ключ сесії (`SECRET_KEY`) генерується автоматично в пам'яті за відсутності файлу `.env`.
* **Динамічна багатомовність (i18n):** Повна локалізація інтерфейсу (англійська за замовчуванням, українська, німецька) з можливістю перемикання мови прямо на сторінці авторизації.
* **Рольова модель доступу:** Розмежування користувачів на Admin, Editor та Viewer із гнучким подетальних налаштуванням прав (читання/запис) на кожну окрему вкладку (категорію) каталогу.
* **Вбудований моніторинг логів:** Зручна адмін-панель для перегляду логів додатка в реальному часі. Підтримує динамічну зміну рівня деталізації (DEBUG, INFO, WARNING, ERROR) без перезапуску контейнера та автоматичну ротацію файлів (макс. 5 МБ) для захисту дискового простору сервера.
* **Резервне копіювання в один клік:** Можливість скачування повного бекапу системи (база даних + іконки) через веб-інтерфейс та швидке гаряче відновлення з ZIP-архіву.
* **Docker-native:** Проект повністю оптимізований для швидкого розгортання через Docker Compose на домашньому сервері чи міні-ПК.
